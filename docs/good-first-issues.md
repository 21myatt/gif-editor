# Good First Issues

These are intentionally small, isolated tasks for new contributors. When filing
them on GitHub, apply the `good first issue` label and link back to this file.

## UI Polish

- Add a `Clear crop` button that resets crop to full source dimensions.
- Add a `Fit to screen` zoom button next to zoom in/out.
- Add tooltip text to the frame range inputs.
- Add a visible keyboard shortcuts help dialog.
- Add a confirmation dialog before `New` clears the current project.

## Transform Filters

- Add an `Invert` transform using the TransformRegistry.
- Add a `Brightness +10%` transform using the TransformRegistry.
- Add a `High contrast` transform using the TransformRegistry.
- Add browser tests for each transform with a 1x1 or 2x1 canvas fixture.

## Export Options

- Add a copy-to-clipboard button for sprite sheet metadata.
- Add a JSON metadata export next to sprite sheet PNG export.
- Add a checkbox to include every Nth frame in sprite sheet export.

## Docs and Tests

- Add screenshots to `README.md` once a stable visual baseline exists.
- Add examples to `CONTRIBUTING.md` for common canvas pixel operations.
- Add accessibility regression cases for dialogs and keyboard shortcuts.

## Suggested GitHub Labels

- `good first issue`
- `help wanted`
- `docs`
- `accessibility`
- `transform`
- `tests`
